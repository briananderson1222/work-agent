# Competitive Scan

Scan the AI agent/IDE landscape for new tools, features, and trends. Cross-reference against Stallion's differentiators.

## Instructions

1. **Read current landscape:**
   - Read `docs/strategy/competitive-landscape.md` for the current snapshot
   - Read `docs/strategy/differentiators.md` for what we're protecting

2. **Search for new developments:**
   - Web search for "AI coding agent 2026", "AI IDE new", "agentic coding tools"
   - Check GitHub trending for AI/agent repositories
   - Search for recent blog posts or announcements from tracked competitors (Codex, Hermes, Pi, T3Code, Happier)
   - Look for new entrants not in our current landscape doc

3. **Analyze findings against our differentiators:**
   - **Threats**: Has a competitor caught up on one of our Tier 1 differentiators?
   - **Opportunities**: Is there a gap in the market that aligns with our roadmap?
   - **New patterns**: Any architectural patterns or UX innovations worth studying?

4. **Update docs:**
   - If new competitor found: add to `docs/strategy/competitive-landscape.md`
   - If steal-worthy feature found: add to the Steal List table
   - Append summary to `docs/strategy/ideation-log.md`

5. **Report to user:**
   - Brief summary of landscape changes
   - Top 3 threats or opportunities
   - Recommended actions (if any)

Run this quarterly or when evaluating strategic priorities.
